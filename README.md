
# Random User

A website which can search hundreds of random user with the Random User API.


## To run locally
1. Clone the repository:
```
git clone git@github.com:dm-valdez/random-user.git
```
2. CD to the project directory
```
cd random-user
```
3. run npm install
```
npm install
```
4. run npm start
```
npm start
```
